# chrome-custom-newtab

크롬 새 탭에 자신의 취향을 반영하고 유용한 기능을 모듈 방식으로 사용할 수 있도록 만들어보고 있습니다.
(2022년 8월~)

### 사용 가능한 모듈

- Weather
- Git Contribution Graph
- D-day Counter
- Life-Quotes
- To-Do-List
- Random Cat Image
- Youtube
- Top-Sites (Chrome-API)
- Shortcut
- letter Counter
